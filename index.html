Biblioteca Digital MHGB
🚀 Sobre o projeto

Este projeto foi desenvolvido para a escola com o objetivo de praticar conceitos de JavaScript, além de aplicar conhecimentos em CSS (estilo), front-end e um pouco de back-end. A aplicação funciona como uma biblioteca virtual, onde é possível cadastrar e visualizar informações sobre livros.

📚 Funcionalidades

✅ Cadastro de livros com os seguintes dados: Nome do livro Autor Editora Imagem da capa Sinopse ✅ Exibição dos livros na categoria "Todos os Livros", em formato de cards. ✅ Visualização da sinopse de cada livro dentro de um modal ao clicar em "Sinopse". ✅ Busca de livros: O usuário digita o nome do livro desejado. Se encontrado, o resultado aparece em um card. Se não encontrado, é exibido um modal avisando que nenhum livro foi localizado.

Sliding Puzzle

Um sliding puzzle 3x3 HTML e CSS JavaScript Sliding Puzzle 3x3 com HTML, CSS e JavaScript Este projeto é uma página web interativa que implementa o clássico Sliding Puzzle 3x3. Desenvolvido com HTML, CSS e JavaScript, o jogo desafia sua lógica e raciocínio. O objetivo é simples: você deve reorganizar as peças numeradas de 1 a 8 em uma grade de 3x3, utilizando o espaço vazio para deslizar as peças até que a ordem correta seja alcançada. É uma ótima forma de passar o tempo e exercitar o cérebro! :)

Como jogar
Clique (ou toque) em uma peça que esteja ao lado do espaço vazio. A peça desliza para o espaço livre. Continue movendo até deixar as peças na sequência 1 → 8 e o vazio no final.
